# FlowSpace

FlowSpace is a full-stack SaaS platform designed for project management and team collaboration. The application combines task management, real-time communication, documentation, reporting, and analytics into a single workspace.

## Features

* Kanban boards with drag-and-drop functionality
* Real-time team chat using Socket.io
* Collaborative documentation and knowledge base
* Calendar and event management
* Analytics dashboard and reporting tools
* File and document management
* Role-based access control (RBAC)
* JWT authentication and authorization
* Multi-tenant architecture for organizations and teams
* Responsive interface with dark/light mode support

## Tech Stack

### Frontend

* React 18
* Vite
* React Router
* Tailwind CSS
* Axios
* Socket.io Client
* TipTap Editor

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Socket.io
* JWT Authentication
* Multer & GridFS
* Helmet
* Express Rate Limit

## Deployment

* Frontend & Backend hosted on Render
* Database hosted on MongoDB Atlas

## Project Goal

Build a scalable SaaS application that centralizes project management, team communication, documentation, and reporting within a single platform.
