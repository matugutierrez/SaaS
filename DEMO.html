FlowSpace

Jira + Slack + Notion — All-in-One Project Management

React · Node.js · Express · MongoDB · Socket.io

📋 Jira — Kanban Boards

Drag & drop tasks across columns. Assign, prioritize, set due dates. Full board management.

💬 Slack — Real-time Chat

Channel-based chat per project. Send messages, see who's online. Powered by Socket.io.

📝 Notion — Wiki & Docs

Rich text editor (TipTap). Version history. Document pages per project.

👥 Multi-tenant

Each organization is isolated. Invite members with a code. Granular RBAC.

🔔 Live Notifications

Real-time notifications for task assignments, comments, and status changes.

📁 File Attachments

Upload files directly to MongoDB (GridFS). No external storage needed.

📊 Productivity Dashboard

Overview of tasks by status, overdue items, assigned tasks, and recent activity.

📜 Audit Log

Every action is logged. Filterable table. Visible to Admin+ and Owner.

🔧 Tech Stack

⚛ React 18 + Vite (frontend)
🖥 Node.js + Express (API)
🗄 MongoDB + Mongoose + GridFS
🔌 Socket.io (real-time)
🔐 JWT + bcrypt (auth)
🎨 Tailwind CSS (UI)
📦 @hello-pangea/dnd (drag & drop)
📝 @tiptap/react (rich text)

👑 Role System

👑 Owner

Full access. Can transfer ownership. Cannot be removed or demoted. Supreme.

⭐ Admin+

Can invite members, manage roles (except Owner), full CRUD, view audit log.

⚡ Admin

Create, edit, delete projects, tasks, boards, documents. Cannot invite or manage roles.

👤 Member

View everything. Chat with team. Cannot create, edit, or delete anything.

🚀 Deploy Steps

1
Create MongoDB Atlas — Free tier (512MB, no credit card). Get your MONGO_URI.
2
Push to GitHub — git init && git add . && git commit -m "init" && git push
3
Connect to Render — New Web Service → select repo. Free plan.
4
Set env vars — MONGO_URI (from Atlas) and JWT_SECRET (any string).
5
Deploy — Render builds automatically. Your app is live at https://your-app.onrender.com
6
Register — First user = Owner. Get invite code from Settings → invite teammates.
